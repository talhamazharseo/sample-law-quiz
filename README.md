# Pakistan Criminal & Civil Law MCQ Quiz

A static, GitHub Pages-ready quiz containing **20 easy multiple-choice questions** about basic criminal and civil law concepts in Pakistan.

Every English question and answer option includes an Urdu translation.

## Quiz Behavior

- The user selects one option for each question.
- The selected answer locks immediately.
- The user cannot change a locked answer.
- No correct or incorrect feedback appears while answering.
- Results and correct answers appear only after all 20 questions are completed.
- The final page shows the score, percentage, and a complete answer review.

## Files

- `index.html` — page content and structure
- `style.css` — responsive English and Urdu design
- `script.js` — questions, answer locking, scoring, review, and sharing
- `.gitignore` — ignores common local editor and operating-system files

## Publish With GitHub Pages

1. Create a new public GitHub repository.
2. Upload every file from this folder to the repository root.
3. Commit the files.
4. Open **Settings → Pages**.
5. Select:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
6. Save.

The published URL normally follows this format:

```text
https://YOUR-USERNAME.github.io/REPOSITORY-NAME/
```
